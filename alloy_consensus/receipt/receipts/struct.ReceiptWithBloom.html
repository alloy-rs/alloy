pub struct ReceiptWithBloom<T = Log> {
    pub receipt: Receipt<T>,
    pub logs_bloom: Bloom,
}
Expand description

Receipt with calculated bloom filter.

This convenience type allows us to lazily calculate the bloom filter for a receipt, similar to Sealed.
